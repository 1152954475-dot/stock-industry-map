把"电池产业链"想成一桶能反复使用的"电力果汁"

电池本质是把电存起来反复用的容器。想象一个果汁桶：

正极是果汁原浆罐（决定能量密度和电压）、负极是空罐子（充电时接收锂离子）、电解液是输送管道（让锂离子来回跑）、隔膜是安全阀（防止正负极接触爆炸）。

充电时，锂离子从正极"游"过电解液、穿过隔膜、住进负极；放电时反着来——这一来一回就是一次充放循环。

📌 这一轮电池行情的核心逻辑：新能源车 + 储能 + AI 数据中心配储 三大需求共振，叠加 锂价上涨带动整条链涨价、钠电/固态新技术放量。

五、记住这条主线2026 年电池板块的投资逻辑

🎯 三条主线，按强弱排序

强主线 1 · 锂价上涨链(弹性最大)：碳酸锂 7万 → 25万 / 吨 → 锂矿(赣锋/天齐) + 正极(湖南裕能/容百) + 电解液(天赐/新宙邦) 全产业链涨价传导。这是业绩弹性王。

强主线 2 · 储能爆发(确定性最强)：AI 数据中心配储 + 风光大基地 → 阳光电源 + 海博思创 + 宁德时代储能板块。订单可见、政策护航。

强主线 3 · 新技术弯道超车(想象空间最大)：钠电池(华阳/传艺) + 固态电池(鹏辉/三祥) + 4680 圆柱。短期题材为主，长期看量产兑现。

📍 三大主线的共同特征：需求确定 + 供给紧张 + 国产领先全球。中国电池产业链全球市占率超 70%，是少有的"全栈国产替代成功"赛道。
