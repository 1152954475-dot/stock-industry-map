# A股产业链全景图

两个生动的HTML5交互页面，帮助投资者快速理解科技和新能源电池产业链。

## 📊 包含内容

### 💻 科技板块全景
- 11个细分领域（PCB、CPU、MLCC、光模块、服务器等）
- 四层产业链金字塔（应用→核心→材料→设备）
- A股龙头股票标注

### 🔋 电池板块全景
- 锂电池工作原理动画演示
- 四层产业链结构（下游应用→电池电芯→四大主材→资源设备）
- 12个细分赛道（正极、负极、电解液、隔膜、锂矿、储能、钠电、固态等）
- 2026投资逻辑与风险提示

## 🚀 特性

- ✅ 纯HTML5 + CSS3 + SVG，无需任何外部依赖
- ✅ 响应式设计，支持手机/平板/PC
- ✅ 丰富的动画效果（充电动画、锂离子流动等）
- ✅ 生动的比喻（造AI服务器、电力果汁桶）

## 📱 访问方式

直接打开 `index.html` 即可查看导航页，点击卡片进入对应板块。
